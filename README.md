# EndemikDB - Aplikasi Katalog Hewan & Tumbuhan Endemik Indonesia

Aplikasi ini dibuat untuk memenuhi tugas UAS Praktikum Pengembangan Aplikasi Mobile. Aplikasi ini menampilkan daftar hewan dan tumbuhan endemik Indonesia dengan fitur database lokal (ROOM), pengambilan data dari API (Retrofit), dan fitur favorit.

## Fitur Utama
- **Splash Screen**: Sinkronisasi data dari API ke database lokal saat pertama kali dijalankan.
- **Home**: Menampilkan daftar hewan dan tumbuhan dalam dua fragmen terpisah.
- **Search**: Mencari data berdasarkan nama.
- **Favorit**: Menyimpan data yang disukai ke database lokal (ikon Hati).
- **Detail**: Menampilkan informasi lengkap termasuk nama latin, keluarga, dan status konservasi.
- **Profil**: Informasi pengembang (Nama, NIM, Foto).

## Sumber Data (API)
Aplikasi ini menarik data secara dinamis dari URL berikut:
- **Link API JSON**: [https://hendroprwk08.github.io/data_endemik/endemik.json](https://hendroprwk08.github.io/data_endemik/endemik.json)

## Teknologi yang Digunakan
- **Bahasa**: Java
- **Database**: Room Persistence Library
- **Networking**: Retrofit & Gson
- **Image Loading**: Glide
- **Navigation**: Android Jetpack Navigation Component
- **UI Component**: Material Design 3 (Support Dark & Light Mode)
